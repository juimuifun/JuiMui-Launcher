กำหนดหน่วยความจำ RAM ที่จัดสรรให้เกมของคุณ ปริมาณที่มากขึ้นจะช่วยเพิ่มประสิทธิภาพของเกม
โปรดทราบว่าไม่ควรจัดสรร RAM มากเกินไป เพราะอาจทำให้โปรแกรมอื่นๆ ช้าลงได้

-RAM สูงสุด: ระบุจำนวน RAM สูงสุดที่จัดสรรให้เกมของคุณ แนะนำว่าไม่ควรเกิน 50% ของหน่วยความจำ RAM ทั้งหมดที่มีในเครื่องคุณ

-RAM ขั้นต่ำ: ระบุจำนวน RAM ขั้นต่ำที่จัดสรรให้เกมเมื่อเริ่มต้น แนะนำว่าไม่ควรเกิน 4GB สำหรับค่าตัวนี้
